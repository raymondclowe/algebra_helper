Key Changes

  • Before: grid-cols-1 sm:grid-cols-2 - 1 column on mobile, causing vertical stacking (1x4 or 1x5 with "I don't know")
  • After: grid-cols-2 - 2 columns on all screen sizes, creating a compact 2x2 grid
  • Benefit: Reduces vertical space used by buttons, preventing overlap with question content
  • Benefit: Better utilizes horizontal screen space on mobile devices
  • Note: Vectors use proper LaTeX notation (\begin{pmatrix}...\end{pmatrix}) with natural line breaking between elements
