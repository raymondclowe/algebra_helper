# IB Math AA Syllabus Coverage Review

**Date:** 2026-01-08  
**Purpose:** Review current coverage of IB Mathematics Analysis & Approaches (SL and HL) syllabus and identify gaps for future expansion

## Executive Summary

The Algebra Helper app currently provides **excellent coverage** of the IB Math AA syllabus, particularly in:
- Core algebra (equations, quadratics, polynomials)
- Advanced HL topics (proofs, complex numbers, matrices, 3D vectors, differential equations)
- Basic calculus (differentiation, integration)
- Probability distributions and hypothesis testing

This review identifies **35 topic areas** for potential enhancement, prioritized by importance to the IB curriculum and alignment with the app's focus on algebraic drill and common operations.

## Current Coverage Analysis

### Strengths (34 levels with extensive coverage)

#### ✅ Excellent Coverage
1. **Number and Algebra** (Levels 1-14, 25-27)
   - Basic arithmetic through advanced sequences
   - Exponentials and logarithms
   - Proof by induction and contradiction
   - Partial fractions
   - Complex numbers (Cartesian and polar forms)

2. **Algebra and Equations** (Levels 5-12)
   - Linear equations through polynomials
   - Quadratics (factoring, solving, discriminant)
   - Factor and remainder theorems

3. **Advanced HL Topics** (Levels 25-34)
   - Proof techniques (induction, contradiction)
   - Matrix algebra (operations, determinants, inverses)
   - 3D vectors (including cross product)
   - Complex polar form and De Moivre's theorem
   - Advanced integration (substitution, parts)
   - Differential equations (separation of variables)
   - Probability distributions (binomial, normal)
   - Hypothesis testing

4. **Calculus** (Levels 19-21, 24-25, 30-32)
   - Differentiation (power rule, chain rule, product rule)
   - Integration basics
   - Critical points

5. **Probability and Statistics** (Levels 21-24, 32-34)
   - Descriptive statistics (mean, median, mode, range)
   - Basic and advanced probability
   - Expected value
   - Distributions and hypothesis testing

### Areas with Room for Enhancement

#### ⚠️ Partial Coverage (needs expansion)
- **Functions:** Missing rational functions, graph transformations, domain/range
- **Trigonometry:** Missing sine/cosine rule, arc length, sector area, reciprocal trig functions
- **Calculus:** Missing tangent/normal lines, quotient rule, definite integrals, optimization
- **Statistics:** Missing standard deviation, variance, correlation, regression
- **Geometry:** Missing 3D solid geometry (volumes, surface areas)

#### ❌ Missing Topics (not yet covered)
- Financial applications (compound interest, depreciation)
- Parallel/perpendicular lines
- Quadratic inequalities
- Modulus equations and inequalities
- Compound angle identities
- Vector and plane equations
- Bayes' theorem
- L'Hôpital's rule
- Maclaurin series
- Volumes of revolution

## Prioritized Recommendations

### Phase 1: High-Priority Core Topics (12 items)

These are fundamental SL/HL topics with strong alignment to algebraic drill:

1. **Financial Applications** (Level 13-14)
   - Compound interest: A = P(1 + r)^t
   - Annual depreciation: V = V₀(1 - r)^t
   - Application problems with time periods

2. **Parallel and Perpendicular Lines** (Level 6-7)
   - m₁ = m₂ for parallel lines
   - m₁ × m₂ = -1 for perpendicular lines
   - Finding equations of parallel/perpendicular lines

3. **Quadratic Inequalities** (Level 10-11)
   - Solving x² + bx + c < 0
   - Using sign diagrams
   - Number line representation

4. **Rational Functions** (Level 14-15)
   - f(x) = 1/x (reciprocal function)
   - f(x) = (ax + b)/(cx + d)
   - Vertical asymptotes: cx + d = 0
   - Horizontal asymptotes: y = a/c

5. **Graph Transformations** (Level 14-15)
   - Translations: f(x) + k, f(x + h)
   - Reflections: -f(x), f(-x)
   - Vertical stretch: af(x)
   - Horizontal stretch: f(ax)

6. **Sine and Cosine Rule** (Level 15-16)
   - Sine rule: a/sin(A) = b/sin(B) = c/sin(C)
   - Cosine rule: c² = a² + b² - 2ab·cos(C)
   - Area of triangle: A = (1/2)ab·sin(C)

7. **Arc Length and Sector Area** (Level 15-16)
   - Arc length: s = rθ (θ in radians)
   - Sector area: A = (1/2)r²θ
   - Application problems

8. **Tangent and Normal Lines** (Level 19-20)
   - Tangent equation: y - y₀ = f'(x₀)(x - x₀)
   - Normal equation: gradient = -1/f'(x₀)
   - Finding equations at given points

9. **Definite Integrals and Area** (Level 24-25 or 30-31)
   - Evaluating ∫[a,b] f(x) dx
   - Area when f(x) < 0 (take absolute value)
   - Area between two curves

10. **Standard Deviation and Variance** (Level 21-22)
    - Calculating σ and σ²
    - Effect of adding/multiplying constants
    - Comparing data sets

11. **Linear Regression and Correlation** (Level 21-22)
    - Pearson's r (correlation coefficient)
    - Regression line: y = ax + b
    - Using regression for prediction
    - Interpretation of a and b

12. **Quadratic Vertex Form** (Level 10-11)
    - f(x) = a(x - h)² + k
    - Identifying vertex (h, k)
    - Axis of symmetry: x = h
    - Converting between forms

### Phase 2: Medium-Priority HL Topics (14 items)

These enhance HL coverage and align well with algebraic operations:

13. **Permutations** (Level 22-23)
    - P(n, r) = n!/(n-r)!
    - Distinguishing from combinations
    - Ordered vs unordered selections

14. **Systems of 3×3 Linear Equations** (Level 28)
    - Gaussian elimination
    - Using determinants
    - Unique, infinite, or no solution cases

15. **Modulus Equations and Inequalities** (Level 7-8 or 14-15)
    - Solving |x| = a
    - Solving |ax + b| = c
    - Solving |x| < a and |x| > a

16. **Sum and Product of Roots** (Level 11-12)
    - For ax² + bx + c = 0: sum = -b/a, product = c/a
    - Finding quadratics from roots
    - Symmetric functions of roots

17. **Reciprocal Trig Functions** (Level 16-17)
    - sec θ = 1/cos θ
    - cosec θ = 1/sin θ
    - cot θ = 1/tan θ
    - Pythagorean identities with sec, cosec, cot

18. **Compound Angle Identities** (Level 16-17)
    - sin(A ± B) = sin A cos B ± cos A sin B
    - cos(A ± B) = cos A cos B ∓ sin A sin B
    - tan(A ± B) = (tan A ± tan B)/(1 ∓ tan A tan B)

19. **Vector and Plane Equations** (Level 28-29)
    - Line: r = a + λb
    - Plane: r · n = a · n
    - Cartesian form: ax + by + cz = d
    - Finding intersections

20. **Bayes' Theorem** (Level 23-24)
    - P(A|B) = P(B|A) · P(A) / P(B)
    - Tree diagram applications
    - Maximum of 3 events

21. **Quotient Rule** (Level 20-21)
    - d/dx[u/v] = (v·u' - u·v')/v²
    - Applications to rational functions

22. **L'Hôpital's Rule** (Level 20-21)
    - Evaluating limits of form 0/0 or ∞/∞
    - Repeated application

23. **Implicit Differentiation** (Level 20-21)
    - Finding dy/dx for equations like x² + y² = r²
    - Product and chain rule applications

24. **Maclaurin Series** (Level 30-31)
    - e^x = 1 + x + x²/2! + x³/3! + ...
    - sin x = x - x³/3! + x⁵/5! - ...
    - cos x = 1 - x²/2! + x⁴/4! - ...
    - Using series for approximations

25. **Volumes of Revolution** (Level 30-31)
    - About x-axis: V = π∫[a,b] y² dx
    - About y-axis: V = π∫[c,d] x² dy

26. **Related Rates** (Level 20-21)
    - Linking rates of change using chain rule
    - Application problems (e.g., expanding circle)

### Phase 3: Lower-Priority Enhancements (9 items)

These add completeness but are less central or harder to implement in text format:

27. **Counterexample Proofs** (Level 26-27)
28. **3D Solid Geometry** (Level 17 or new 17.5)
29. **Ambiguous Case of Sine Rule** (Level 16-17)
30. **Quadratic Trigonometric Equations** (Level 16-17)
31. **Trig Function Symmetry** (Level 16-17)
32. **Kinematics with Calculus** (Level 20-21 or 28-29)
33. **Continuous Random Variables** (Level 32-33)
34. **Euler's Method for DEs** (Level 31-32)
35. **Line-Plane Intersections** (Level 28-29)

## Implementation Strategy

### Guiding Principles

1. **Focus on Algebraic Operations**: Prioritize topics that involve manipulation and calculation rather than graphical interpretation
2. **Multiple-Choice Format**: All questions must work in MC format with LaTeX rendering
3. **Common Operations**: Emphasize frequently-used techniques that students need fluency in
4. **Incremental Addition**: Add 3-5 new question types per level to avoid overwhelming students
5. **Maintain Balance**: Keep spaced repetition and "why" question features intact

### Recommended Workflow

#### Phase 1 Implementation (High Priority - 12 topics)
**Target:** Add to existing levels over 3-4 iterations

1. **Iteration 1:** Financial applications, parallel/perpendicular lines, quadratic inequalities
   - Levels affected: 6-7, 10-11, 13-14
   - Estimated: 15-20 new question types

2. **Iteration 2:** Rational functions, graph transformations, vertex form
   - Levels affected: 10-11, 14-15
   - Estimated: 18-25 new question types

   - Levels affected: 15-16
   - Estimated: 12-15 new question types

4. **Iteration 4:** Tangents/normals, definite integrals, standard deviation, regression
   - Levels affected: 19-20, 21-22, 24-25
   - Estimated: 20-25 new question types

#### Phase 2 Implementation (Medium Priority - 14 topics)

Focus on topics 13-26, adding 3-4 new question types per level

#### Phase 3 Implementation (Lower Priority - 9 topics)

## Question Type Examples

### Example 1: Financial Applications (High Priority)

**Question Type A: Compound Interest**
```
LaTeX: A bank offers 5% annual interest compounded yearly. 
If £1000 is invested, what will the value be after 3 years?

Formula: A = P(1 + r)^t

Options:
A) £1150.00
B) £1157.63  ← Correct
C) £1160.00
D) £1200.00
```

**Question Type B: Depreciation**
```
LaTeX: A car worth £20000 depreciates at 15% per year. 
What is its value after 2 years?

Formula: V = V₀(1 - r)^t

Options:
A) £13600.00
B) £14000.00
C) £14450.00  ← Correct
D) £17000.00
```

### Example 2: Parallel/Perpendicular Lines (High Priority)

**Question Type A: Parallel Line**
```
LaTeX: Find the equation of a line parallel to y = 3x + 2 
passing through (1, 5).

Options:
A) y = 3x + 2  ← Correct
B) y = 3x + 5
C) y = -⅓x + 5⅓
D) y = x + 4
```

**Question Type B: Perpendicular Line**
```
LaTeX: Find the gradient of a line perpendicular to y = 2x - 1.

Options:
A) 2
B) -2
C) ½
D) -½  ← Correct
```

### Example 3: Sine Rule (High Priority)

**Question Type A: Finding a Side**
```
LaTeX: In triangle ABC, A = 30°, B = 45°, and a = 10. 
Find b using the sine rule.

Formula: a/sin(A) = b/sin(B)

Options:
A) 10√2  ← Correct
B) 10
C) 5√2
D) 14.14
```

**Question Type B: Finding an Angle**
```
LaTeX: In triangle ABC, a = 8, b = 10, and A = 30°. 
Find sin(B).

Options:
A) 0.625  ← Correct
B) 0.5
C) 0.8
D) 1.25
```

### Example 4: Tangent Line (High Priority)

```
LaTeX: Find the equation of the tangent to y = x² at x = 3.

Steps:
1. Find y-coordinate: y = 9
2. Find gradient: dy/dx = 2x, so m = 6 at x = 3
3. Use point-slope form: y - 9 = 6(x - 3)

Options:
A) y = 6x - 9  ← Correct
B) y = 6x + 9
C) y = 2x + 3
D) y = 3x
```

### Example 5: Standard Deviation (High Priority)

```
LaTeX: Data set A: {2, 4, 6, 8, 10}. 
If each value is multiplied by 3, how does the standard deviation change?

Options:
A) Stays the same
B) Multiplied by 3  ← Correct
C) Multiplied by 9
D) Divided by 3
```
